# 4. Формат IP-адреса: классовая адресация. Особые IP-адреса: неопределённый адрес, адрес узла, ограниченный широковещательный, широковещательный, обратная петля.

## Классовая адресация.

Изначально адресация в IP-сетях основывалась на классах: первые биты определяли
класс сети, а по классу сети можно было определить, сколько битов присвоено
номеру сети и номеру узла, что в общей сложности составляло 5 классов: A, B, C,
D, E.

Характеристики классов:
| Класс | Старшие биты | Назначение             | Диапазон первого октета | Маска по умолчанию | Число сетей / узлов в сети    |
| ----- | ------------ | ---------------------- | ----------------------- | ------------------ | ----------------------------- |
| A     | 0            | Крупные сети           | 1 – 127                 | 255.0.0.0          | 126 сетей, ~16.7 млн узлов    |
| B     | 10           | Средние сети           | 128 – 191               | 255.255.0.0        | ~16.3 тыс. сетей, 65 534 узла |
| C     | 110          | Малые сети             | 192 – 223               | 255.255.255.0      | ~2.1 млн сетей, 254 узла      |
| D     | 1110         | Многоадресная рассылка | 224 – 239               | Не определена      | Для групповой рассылки        |
| E     | 1111         | Зарезервирован         | 240 – 255               | Не определена      | Зарезервированы               |

## Особые IP-адреса:

### Неопределённый адрес

0.0.0.0 – адрес по умолчанию. Он используется для ссылки на отсутствие адресата.
Применяется в качестве указателя для сервера на принятие информации от любых IP.

### Адрес узла

Возьмем адрес 111.22.333.1 в сети класса B (маска по умолчанию
255.255.0.0).  Сетевая часть: 111.22 Узловая часть: 333.1 Полный адрес
111.22.333.1 и является адресом узла, исключая особые адреса такие, как
широковещательный и тп.

### Ограниченный широковещательный адрес

255.255.255.255 - все двоичные разряды IP-адреса равны 1, пакет с таким адресом
назначения должен рассылаться всем узлам, находящимся в той же сети, что и
источник этого пакета, не пересекает маршрутизаторы.

## Широковещательный адрес

N.N.N.255 - в поле номера узла назначения стоят только единицы, пакет, имеющий
такой адрес, рассылается всем узлам сети с заданным номером сети. Например, в
сети 192.168.5.0 с маской подсети 255.255.255.0 пакет с адресом 192.168.5.255
доставляется всем узлам этой сети.

### Обратная петля

127.0.0.0 – localhost. Идентификатор, который указывает сам на себя. Если
устройству требуется обратиться по IP к себе, будет использоваться
соответствующая запись.
