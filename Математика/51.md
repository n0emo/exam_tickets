# Разложение основных элементарных функций в ряд Маклорена.

## Вывод ряда Маклорена для функции

Для примера разложим функцию $e^x$ в ряд Тейлора в точке $x=0$ (то есть в ряд
Маклорена):

$$f^{(n)}(x) = (e^x)^{(n)} = e^x; f^{(n)}(0) = 1$$

Получаем ряд:

$$\sum_{n=0}^\infty \frac{1}{n!} x^n$$

Все производные $e^x$ ограничены на любом отрезке $[-a; a]$, то есть 
$|f^{(n)}(x)| = e^x \le M = e^a$. Поэтому согласно о теореме о разложении
можно записать:

$$e^x = \sum_{n=0}^\infty \frac{1}{n!} x^n$$

Радиус сходимости вычислим по формуле Даламбера.

$$R = \lim \left| \frac{a_n}{a_{n+1}} \right| = 
\lim \frac{(n+1)!}{n!} = \lim \frac{(n+1) \cdot n!}{n!} =
\lim (n+1) = +\infty$$

Тогда интервал сходимости ряда -- $(-\infty; +\infty)$

## Разложения основных функций

Следующие разложения и интервалы их сходимости следует запомнить.

$$1.\ e^x=\sum_{n=0}^\infty\frac{x^n}{n!}, x \in (-\infty;+\infty)$$
$$2.\ \sin{x}=\sum_{n=0}^\infty\frac{x^{2n-1}}{(2n-1)!},x\in (-\infty;+\infty)$$
$$3.\ \cos{x}=\sum_{n=0}^\infty\frac{x^{2n}}{(2n)!},x\in (-\infty;+\infty)$$
$$4.\ (1+x)^m=\sum_{n=0}^\infty\frac{m(m-1)...(m-n+1)}{n!}\cdot x^n,x\in(-1;1)$$
$$5.\ \frac{1}{1+x}=\sum_{n=0}^\infty(-1)^n x^n,x\in(-1;1)$$
$$6.\ \frac{1}{1-x}=\sum_{n=0}^\infty x^n,x\in(-1;1)$$
$$7.\ \ln(1+x)=\sum_{n=0}^\infty\frac{(-1)^{n+1}}{n}\cdot x^n,x\in(-1;1)$$
