# 23 Методы подтверждения надежности

Основные методы подтверждения надежности:

- **расчетный метод** определения надежности: Метод, основанный на вычислении
  показателей надежности по справочным данным о надежности компонентов и
  комплектующих элементов объекта, по данным о надежности объектов-аналогов, по
  данным о свойствах материалов и другой информации, имеющейся к моменту расчета
  надежности
- **расчетно-экспериментальный** метод определения надежности: Метод оценки
  надежности объекта путем расчета, при котором показатели надежности всех или
  некоторых составных частей объекта определены экспериментально
- **экспериментальный метод** определения надежности: Метод оценки показателей
  надежности путем статистической обработки данных, полученных при испытаниях или
  эксплуатации объекта в целом
