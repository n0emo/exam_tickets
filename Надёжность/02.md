# 2. Состояния изделий, переходы из одного состояния в другое. Трактовка отказа и неисправности, понятие критичного отказа

## Состояния изделия

| Состояние         | Определение                                                                                                                               | Англ.         |
| ----------------- | ----------------------------------------------------------------------------------------------------------------------------------------- | ------------- |
| Исправное         | Соответствует всем требованиям, установленным в докумнентации                                                                             | perfect       |
| Неисправное       | Не соотвутствует хотя бы одному из требований, установленных в документации                                                               | imperfect     |
| Работоспособное   | Способен выполнять требуемые функции                                                                                                      | up            |
| Неработоспособное | Не способен выполнить хотя бы одну требуемую функцию по причинам, зависящим от него                                                       | down          |
| Рабочее           | Выполняет какую-либо требуемую функцию                                                                                                    | operating     |
| Нерабочее         | Не выполняет ни одной из требуемых функций                                                                                                | non-operating |
| Предельное        | Дальнейшая эксплуатация недопусима или нецелесообразна, либо восстановление его работоспособного состояния невозможно или нецелесообразно | limiting      |
| Опасное           | Недопустимый риск причинения вреда людям или окрудающей среде, существенных материальных потерь или других неприемлемыз последствий       | hazardous     |
| Техническое       | Характеризуется совокупностью установленных в документации параметров, описывабщих его способность выполнть требуемые функции             | technical     |
| Предотказное      | Повышенный риск отказа                                                                                                                    | predfault     |

## Понятия отказа, неисправности, критического отказа

**Отказ**: событие, заключающееся в нарушении работоспособного состояния объекта.
Отказ может быть полным или частичным. Полный отказ характеризуется переходом
объекта в неработоспособное состояние. Частичный отказ - в частично неработоспособное.

**Неисправность**: нарушение требоания, установленное в документации. При этом,
неисправное изделие может быть работоспособном; исправное изделие может быть
неработоспособным.

**Критичный отказ**: отказ, при котором изделие переходит в опасное состояние,
т.е возможен риск причинения вреда людям или окружеющей среде и т.д.
